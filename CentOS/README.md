# 安装系统

## 1.SSH 连接 VPS，并进入 boot 文件夹，下载 Netboot 启动文件
  
### CentOS arm64

```bash
yum update -y
yum apt
cd /boot/efi/EFI
wget https://github.com/netbootxyz/netboot.xyz/releases/download/2.0.74/netboot.xyz-arm64.efi
```
  
### x86_64

```bash
cd /boot/efi/EFI
wget https://boot.netboot.xyz/ipxe/netboot.xyz.efi
```
